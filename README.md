# Laravel 7 Formulario Simples com Login

## Instalando

commandos:

```

- git clone <PROJECT_URL>
- composer update
- adicionar .env na raiz e mudar as configurações do seu banco
- php artisan migrate:fresh --seed
- php artisan serve

```
